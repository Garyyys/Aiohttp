Если хотим легко и просто
1)docker compose-up - собирает все обзары и поднимает контейнера с авто установлением связи между контейнерами
2)Не забыть создать таблицу и добавить запись, иначе будет ругаться, что такого параметка не существует.

Если хотим все усложнить
Перед этим надо зайти в директории db и app  и пробилдить контейнера  ( docker build . )
1)заходим в директорию контейнера(db) docker run -e POSTGRES_PASSWORD=123 -p 127.0.0.1:8081:5432 <cache, берем значение после билда>
2)заходим в деррикторию app  docker run -p 127.0.0.1:8080:8080 <cache, берем значение после билда>
3)создаем свою user-defined bridge network(для возможности контейнерам видеть друг друга) docker network create <name of network>
4)подключаем наши контейреа в только что созданную сеть - docker network connect <name of network> <name of container>
